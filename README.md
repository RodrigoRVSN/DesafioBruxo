## üöÄ Installing DesafioBruxo

#### Clone repository

```powershell
  git clone https://github.com/RodrigoRVSN/DesafioBruxo.git
```

#### Install dependencies

```powershell
  yarn install
```

#### Run application

```powershell
  yarn dev
```

#### Run unit tests

```powershell
  yarn test:watch
```

#### Run tests E2E

```powershell
  yarn cy:run
```
